Keep the crop window, dimensions, and recorder position identical for every
export. In the source drawings, gray fingers are lifted/open and white fingers
are contacting/closed.

The complete-pose image is the visual reference for hand placement. Exact
half-thumb and partial double-hole states are rendered by the functional SVG
fingering map, because those contact details are not always visible in a front
pose. Never infer a missing half or partial state from finger color alone.
